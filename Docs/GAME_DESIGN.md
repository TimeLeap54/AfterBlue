# AfterBlue Game Design

AfterBlue is a short 3D atmospheric fishing collection game where the player fishes from a small boat above a flooded civilization and records fish, mutations, and submerged scenery in a journal.

## Core Rule

AfterBlue is not a game about exploring a huge ocean. It is a game about compressing scenes of a flooded world into focused fishing points.

## Player Fantasy

The player is the last angler drifting over a submerged world in a small boat, catching ordinary fish, mutated fish, and recording flooded landmarks.

## Core Loop

- 30 seconds: cast bobber -> wait for bite -> timed input -> reel -> catch fish
- 3 minutes: fish several times -> discover new fish -> sell duplicates -> gain money and records
- 15 minutes: buy bait or rod -> target rare fish -> fill records -> unlock next area
- Full game: flooded village -> transmission tower waters -> sunken cathedral/lighthouse -> legendary fish -> ending
