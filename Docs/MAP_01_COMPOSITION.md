# MAP_01 Composition Lock

The map should be discovered by landmark chaining, not by minimap or forced quest path.

## Sightline Chain

Start / Supply Buoy -> H1 roof and weeds -> Central submerged road -> H2 tilted traffic light -> H3 large debris silhouette -> Return buoy light.

Only the current and next landmark should read strongly in normal play. The whole map should not be visible as one complete board.

## Greybox Choice

Day1 uses Layout A: Landmark Chain.

Reason: it best supports free movement while still giving the player a readable next place to investigate.

## Reference Direction

The current target follows the Map_01 production reference set:

- Start sits at the lower-left supply buoy and must remain visually simple.
- H1 is the bright shallow residential cluster.
- Central Road is the broad readable navigation connector.
- H2 is the hero traffic-light/crosswalk composition.
- H3 is the darker deep debris field.
- Open water between these zones is intentional and should not be filled just because the editor view feels empty.
