# MAP_01 Production Lock

S2 locks the production direction for Map_01 as the single launch map. See `Docs/MAP_01_PRODUCTION_BIBLE.md` for the full production bible.

## Scale

- Original planning layout: 64 x 44
- Current launch-map layout: 192 x 132
- Playable area: X -90 to 90, Z -60 to 60
- World scale: 4.0
- Approximate world read: 768 x 528

## Density Targets

- Open navigation water: 45-55%
- Habitat mid-density areas: 20-25%
- Underwater structure and decor: 15-20%
- Hero landmark density: 5-8%
- Outer silhouettes: 5-10%

## Asset Priority

Hero assets:

- Small wooden boat
- Seated angler
- Start supply buoy
- H2 tilted traffic light
- H1 representative flooded roof/house top
- H3 representative deep debris structure
- Legendary fish silhouette

Modular assets:

- Roads, crosswalks, roofs, poles, signs, windows, weeds, rocks, metal frames, planks, pipes, crates

Background/filler:

- Distant building masses, distant roofs, far poles, low-detail debris silhouettes

## Do Not Finalize During S2

- Final water shader
- Final camera feel
- Final lighting/volume values
- Full art-pass asset set
- Final fish biome economy

S2 verifies map scale, free navigation, landmark spacing, and production scope.
