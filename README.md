# AfterBlue

AfterBlue is a small 3D atmospheric fishing collection game set above a flooded civilization.

The project goal is to finish a small Steam-release candidate while practicing Blender asset production and Codex-assisted development.

Core sentence:

> AfterBlue is not a game about exploring a huge ocean. It is a game about compressing scenes of a flooded world into focused fishing points.

## Project Setup

- Engine: Unity 2022.3 LTS
- Render pipeline: Universal Render Pipeline
- Main project folder: `C:\Users\minil\GameMaking\AfterBlue`
